# Minsc VSCode Extension

VSCode Extension for Bitcoin's [Minsc Programming language](https://min.sc/)

## Features

- [x] Syntax highlighting
- [x] Comment Toggling
- [x] Auto closing
- [] Auto complete
- [x] Code Snippets
- [] Hover messages
- [] Compile to Script Opcodes
- [] Analyze Minsc code
- [] Linting & Error Messages
- [] Semantic highlighting
