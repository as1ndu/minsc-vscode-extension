# Minsc VSCode Extension

VSCode Extension for Bitcoin's [Minsc Programming language](https://min.sc/)

## Features

- [x] Syntax highlighting
- [x] Comment toggling
- [x] Auto closing
- [x] Auto complete
- [x] Code Snippets
- [x] Hover Messages
- [x] Icon support for .minsc files
- [ ] Compile to Script Opcodes
- [ ] Analyze Minsc code
- [ ] Linting, Warning & Error Messages
